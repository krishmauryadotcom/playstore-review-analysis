# Play Store App Review Analysis

This project focuses on analyzing user reviews from Play Store applications to understand customer sentiment, rating patterns, and common user concerns. The objective was to clean raw review data, perform analysis, and generate meaningful insights using Python.

Project Objective

To explore app review data and identify:

- Overall customer sentiment
- Relationship between ratings and review text
- Common positive and negative feedback
- Trends in user satisfaction
- Areas where apps can improve

 Tools & Libraries Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- NLTK
- TextBlob
- Google Colab

 Work Performed

 Data Cleaning
- Removed null values
- Removed duplicate records
- Standardized text data
- Handled inconsistent entries

Exploratory Data Analysis
- Rating distribution analysis
- Review count trends
- Comparison of ratings vs sentiment
- Category-wise observations

Sentiment Analysis
- Classified reviews as Positive, Negative, or Neutral
- Used NLP libraries for text processing
- Measured sentiment trends across records

 Data Visualization
Created charts for:

- Rating distribution
- Sentiment breakdown
- Review trends
- Comparative analysis

 Key Insights

- Higher-rated apps generally received more positive feedback
- Low ratings were often linked to bugs, ads, or performance issues
- Sentiment analysis helped summarize thousands of reviews quickly
- Visual dashboards improved understanding of customer behavior
